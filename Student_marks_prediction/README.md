# Student Marks Prediction using Linear Regression

## Project Overview
This project predicts students' marks using the Linear Regression machine learning algorithm. It analyzes the relationship between study-related features (such as study hours, attendance, or other input variables) and students' academic performance to estimate their final marks.

## Dataset
- Source: Kaggle
- Number of rows:100
- Number of columns:3

## Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Workflow
- Data Loading
- Exploratory Data Analysis (EDA)
- Data Cleaning
- Feature Selection
- Train-Test Split
- Model Training using Linear Regression
- Model Evaluation
- Prediction

## Model Evaluation
- R² Score:0.9462669897223311
- Mean Absolute Error (MAE):2.9582049931174894
- Mean Squared Error (MSE):11.332501955033276
- Root Mean Squared Error (RMSE):3.366378165778954

## Results
The Linear Regression model predicts student marks based on the input features. The model's performance is evaluated using metrics such as R² Score, MAE, MSE, and RMSE to measure prediction accuracy.
